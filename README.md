# feed-irc-bot

将论坛和维基的 Atom 动态转发到 IRC。

> 不论天荒与地老，我们 IRC 上见。

## 安装与运行

```
sudo zypper install git nodejs

git clone https://github.com/openSUSE-zh/feed-irc-bot.git

cd feed-irc-bot

npm install --no-optional # avoid some troublesome libraries

npm start
```
